# Hunted by Kuntilanak

A horror game built using the **Ursina Engine**.

## Description
Experience the terror of being hunted by Kuntilanak in this indie horror game. Explore the environment, stay alive, and uncover the mystery.

## Features
- Immersive horror atmosphere.
- Ghost AI (Kuntilanak).
- Items and inventory system.
- Interactive environment.

## How to Run

### Prerequisites
- Python 3.x
- Ursina Engine

### Installation
1. Clone this repository.
2. Install dependencies:
   ```bash
   pip install ursina
   ```
3. Run the game:
   ```bash
   python main.py
   ```

## Controls
- **WASD**: Move
- **Shift**: Run
- **E**: Interact
- **F**: Flashlight
- **Q**: Toggle Flashlight
- **Esc**: Exit
