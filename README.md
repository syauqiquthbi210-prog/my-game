# Hunted by Kuntilanak

Game horor yang dibuat menggunakan **Ursina Engine**.

## Deskripsi
Rasakan kengerian dikejar oleh Kuntilanak dalam game horor indie ini. Jelajahi lingkungan, bertahan hidup, dan ungkap misterinya.

## Fitur
- Atmosfer horor yang imersif.
- AI Hantu (Kuntilanak).
- Sistem item dan inventaris.
- Lingkungan interaktif.

## Cara Menjalankan

### Prasyarat
- Python 3.x
- Ursina Engine

### Instalasi
1. Clone repositori ini.
2. Instal dependensi:
   ```bash
   pip install ursina
   ```
3. Jalankan game:
   ```bash
   python main.py
   ```

## Kontrol
- **WASD**: Bergerak
- **Shift**: Berlari
- **E**: Interaksi
- **F**: Senter
- **Q**: Nyalakan/Matikan Senter
- **Esc**: Keluar
